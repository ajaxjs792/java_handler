The example demonstrates how to use SCSS in a Java web application with the `libsass-maven-plugin`.

This is the project structure:

```
pom.xml
src
├───main
│   ├───java
│   │   └───com
│   │       └───zetcode
│   │           ├───bean
│   │           │       City.java
│   │           ├───service
│   │           │       CityService.java
│   │           │       ICityService.java
│   │           └───web
│   │                   GetCities.java
│   ├───resources
│   ├───sass
│   │       style.scss
│   └───webapp
│       │   index.jsp
│       │   showCities.jsp
│       └───css
│               style.css
└───test
    └───java
```
